# real-estate

**"사용자가 부동산 매물을 쉽게 확인할 수 있는 게시판입니다."**
-----
### 개발기간

2023. 06~ 2023. 07 (4주)
-----
### 개발환경
- java17
- Springboot
- Maven
- MariaDB
- CSS3
- HTML5
- Security5
- Thymeleaf
---
### 주요기능
- 로그인, 회원가입
- 관리자 role 로그인 시 게시글 작성, 수정, 삭제 가능 / 유저 role 로그인 시 조회만 가능
- 이미지 업로드
- 게시글 10개 마다 페이징
